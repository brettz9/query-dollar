# query-dollar

Ultra-light, ESM-only wrapper for `document.querySelector` and
`document.querySelectorAll`.

## Install

```sh
npm i query-dollar
```

## Usage (bundler)

```js
import {$, $$} from 'query-dollar';
```

## Usage (direct usage)

```js
import {$, $$} from './node_modules/query-dollar/dollar.js';
```
